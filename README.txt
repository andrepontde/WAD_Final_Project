To run the application:

Navigate to the backend folder: Open your terminal or command prompt and navigate to the backend directory.

Start the server: Run the following command to start the Node.js server:

npm install -> This is done to install all of the express dependencies, and other node modules used by the server - this is done so the final submission does not include the node modules folder.
npm start 

Access the application:
Open your web browser and go to http://localhost:3000 to access the application.

The GitHub repository is: https://github.com/andrepontde/my-first-web-app/tree/main
